# Input 输入框 测试

## 何时使用

- 需要用户输入表单域内容时。
- 提供组合型输入框，带搜索的输入框，还可以进行大小选择。

## 基本使用

:::demo 基本使用。

```js
const Demo= () =>{
  return(<Input placeholder="Basic usage" />)
}
```
:::
